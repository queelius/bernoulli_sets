\begin{abstract}
We introduce the \emph{Bernoulli set model}, a compositionally closed probabilistic framework for \emph{random approximate sets}---sets that approximate an objective set with independent, element-wise errors parameterized by false positive and false negative rates.
The central property of the model is \emph{rate computability}: every set-theoretic operation (complement, union, intersection, difference) on Bernoulli sets yields a (possibly higher-order) Bernoulli set whose error rates are closed-form functions of the operand rates and set cardinalities, and this computability extends to arbitrary compositions.
From two axioms---element-wise independence and conditional independence of error rates---we derive the probability distributions of all standard binary classification measures (precision, recall, etc.), the joint entropy of error counts, and the monoidal structure of approximate sets under union and intersection.
The framework unifies the analysis of probabilistic data structures such as Bloom filters and perfect hash filters: any implementation satisfying the Bernoulli axioms inherits the full theory as an abstract data type.
An interval arithmetic extension propagates uncertain rates through set operations, yielding guaranteed bounds on output error rates.
\end{abstract}
